Projects

Employee Management System

The Employee Management System is a full-stack project designed to streamline employee data management. It allows administrators to perform CRUD operations—Create, Read, Update, and Delete—on employee records efficiently. With an intuitive frontend and a robust backend, it ensures accurate and accessible data management. The system improves organizational productivity by centralizing employee data handling. Additionally, it offers secure data storage and retrieval to protect sensitive information.

Amazon Clone

The Amazon Clone Project is a frontend website that replicates the official Amazon e-commerce site. Using HTML and CSS, the project recreates the look and feel of Amazon, showcasing a responsive and visually appealing interface. It includes key features such as product listings, navigation menus, and user interface elements. This project demonstrates proficiency in web design and attention to detail. It serves as a valuable practice in frontend development and design principles.

Flipkart Clone

The Flipkart Clone Project is a frontend website that replicates the official Flipkart e-commerce site. Developed using HTML, CSS, and JavaScript, this project recreates the design and functionality of Flipkart, including interactive login and signup forms. It offers a responsive and visually appealing user interface, featuring product listings, navigation menus, and various UI elements. This project showcases skills in web design, frontend development, and JavaScript interactivity, demonstrating attention to detail and design accuracy.

Rock-Paper-Sessior Game

The Rock-Paper-Scissors Game Project is a frontend application developed using HTML, CSS, and JavaScript. This interactive game allows users to play rock-paper-scissors against the computer, with scores automatically updated and displayed in real-time. The project features a user-friendly interface and smooth gameplay mechanics. It highlights proficiency in web design, frontend development, and JavaScript programming, demonstrating the ability to create engaging and dynamic web applications.
